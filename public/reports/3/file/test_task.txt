Тестовое задание.

Необходимо создать систему для решения математических уравнений(линейных, квадратных)

- Фронтенд - это SPA, написанное на react. Позволяет пользователю ввести данные уравнения.
  Отображает результат, полученный от бекенда. Для верстки используте flexbox.
- Бекенд - это ruby или elixir приложение, которое получает данные уравнения
  от фронтенда, решает его и отправляет результат обратно.
- Протокол взаимодействия фронтенда и бекенда: WebSocket
- Вы можете использовать дополнительные фреймворки и/или библиотеки, если считаете это нужным.
- Вопросы валидации, форматы ввода данных уравнений, вывода результата остаются на ваше усмотрение.
  Желательно, чтобы система была "юзабельной".
- Жестких ограничений по времени нет, но перед началом выполнения тестового задания оцените его по 
  времени и отправте оценку нам. Если в процессе выполнения оценка меняется, то просим вас сообщать 
  об этом.
- Если требуются какие-либо уточнения по тестовому заданию, то задавайте их на любом этапе
  выполнения.
- Если вы передумаете делать тестовое задание сразу после прочтения условия или в процессе
  выполнения то сообщите об этом нам, пожалуйста.

Обратите внимание на следующие вопросы:
- Тесты следующих уровней, которые покрывают выбранные вами крайние случаи.
  ui, которые тестируют, что приложение в целом функционирует.
  unit тесты, которые тестируют классы/модули/фунцкии, предназначенные для решения уравнений.
- Предусмотрите возможность наиболее "безболезненного" добавления нового типа уравнения (чтобы
  добавить новый тип уравнения необходимо "трогать" меньше частей приложения).
